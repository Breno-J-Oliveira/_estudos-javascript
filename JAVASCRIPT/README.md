## 📖 Sobre

Primeiro contato com JavaScript. Introdução aos fundamentos essenciais da linguagem: declaração de variáveis, saída de dados via `console.log`, tipos primitivos e interação com o usuário através de `alert()` e `prompt()`.

## 🧠 Conceitos Abordados

### DIA 1 — Primeiros Passos
- Inclusão da tag `<script>` em páginas HTML
- Declaração de variáveis com `var`
- Caixas de diálogo: `alert()`, `prompt()`
- Saída no console: `console.log()`
- Concatenação de strings

### DIA 1 2.0 — Aprimoramento
- Leitura de inputs via `document.form.field.value`
- Manipulação de formulários HTML
- Conversão de tipos (`parseInt`, `parseFloat`)
- Operações matemáticas básicas

## 💻 Exemplos de Código

```javascript
// Exemplo 1: Saudação com prompt
let nome = prompt("Qual é o seu nome?");
console.log("Olá, " + nome + "!");
alert("Bem-vindo, " + nome);

// Exemplo 2: Soma de dois números
let num1 = parseInt(prompt("Primeiro número:"));
let num2 = parseInt(prompt("Segundo número:"));
document.write("Resultado: " + (num1 + num2));
```
