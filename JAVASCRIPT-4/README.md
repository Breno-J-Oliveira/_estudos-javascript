📘 Sobre este README

Este projeto documenta uma atividade prática de manipulação de Arrays em JavaScript, com foco na criação de interações dinâmicas utilizando botões e atualização do conteúdo na página.

O objetivo foi desenvolver uma aplicação simples capaz de:

Criar e manipular um Array

Exibir dados dinamicamente no HTML

Adicionar e remover elementos da lista

Trabalhar com funções em JavaScript

Manipular o DOM com getElementById()

Criar interações usando eventos onclick

🧭 Resumo da Atividade
🔹 Exemplo de Array em JavaScript
🎯 O que faz:

A aplicação inicia com uma lista de frutas pré-definida:

["Maçã", "Banana", "Laranja"]


O usuário pode:

📌 Visualizar as frutas da lista

➕ Adicionar uma nova fruta via prompt()

➖ Remover a última fruta da lista

🔄 Atualizar automaticamente o conteúdo exibido na página

⚙️ Funcionalidades Implementadas
✅ Mostrar Frutas

Exibe todas as frutas armazenadas no array utilizando:

.join("<br>")

innerHTML

Atualização dinâmica da página

✅ Adicionar Fruta

Solicita o nome da nova fruta com prompt()

Adiciona ao array usando .push()

Atualiza automaticamente a exibição

✅ Remover Última Fruta

Remove o último item com .pop()

Atualiza o conteúdo da página

Valida se o array está vazio

🧠 Conceitos Trabalhados

Estrutura de dados Array

Métodos .push() e .pop()

Método .join()

Manipulação do DOM

document.getElementById()

innerHTML

Funções em JavaScript

Interação com prompt()

Eventos com onclick
